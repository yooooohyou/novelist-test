# 《顶级气运，悄悄修炼千年》风格参考手册

## 一、核心风格标签
**系统流 + 扮猪吃虎 + 苦修龟缩 + 好感度游戏 + 轻松搞笑**

---

## 二、叙事视角与声音

### 视角
- **第三人称有限视角**，以主角为中心
- 偶尔切换到配角/反派视角（场景切换时），用 `……` 和 `另一边。` 过渡
- 敌方视角主要用于制造反差笑果（如敌人震惊、恐惧）

### 内心独白
- **极其高密度**，几乎每段都有主角的内心吐槽
- 直接以叙述文体呈现（非引号对话），如：
  - `韩绝心中破口大骂。`
  - `韩绝默默想到。`
  - `绝壁是毒药！`
- 吐槽风格：网络用语 + 现代人思维 + 修仙世界的反差
- 经常用短句独占一行表达情绪爆发

### 叙述语言
- **极其口语化**，零文艺腔
- 短句为主，很少出现超过30字的长句
- 大量单句成段
- 感叹号使用频繁
- 避免任何环境描写的堆砌

---

## 三、语言特征清单

### 必用元素
| 类型 | 示例 |
|------|------|
| 网络吐槽 | "艹！"、"我擦"、"玛德"、"尼玛"、"就这？"、"可以的！" |
| 自嘲式感叹 | "我这该死的魅力！"、"老子不信了！" |
| 反差独白 | 外表恭敬 → 内心："臭婆娘，你配得上我？" |
| 短句爆发 | "不行！"（独占一行）、"来了！"、"稳了。" |
| 切了一声 | `韩绝切了一声，完全不动心。` |
| 气抖冷 | `韩绝气抖冷。`（气得发抖浑身发冷） |
| 啧啧称奇 | `韩绝啧啧称奇` |

### 禁用元素
| 避免 | 原因 |
|------|------|
| 华丽辞藻 | 不符合口语化风格 |
| 大段环境描写 | 破坏节奏 |
| 诗词歌赋 | 不是传统仙侠 |
| 文言文对话 | 角色说话偏现代 |
| 心理分析式叙述 | 用直白吐槽代替 |

## 四、系统面板格式

### 属性面板（【】框）
```
【姓名：韩绝】
【寿命：16/65】
【种族：凡人】
【修为：无】
【功法：六道轮回功（可传承）】
【法术：无】
【灵根资质：六道灵体，蕴含顶级风、火、水、土、木、雷灵根】
```

### 选择系统
```
【你遭遇某某的威胁，你有以下选择】
【一，选项A，可获得XXX】
【二，选项B，可获得YYY】
```
- 主角几乎总是选择**低调/安全**的选项
- 选项设计：一个高调高风险高回报，一个低调安全有隐藏好处
- 拒绝高调选项的过程本身就是笑点

### 好感度系统
```
【某某对你产生好感，当前好感度为X星】
【某某对你的好感增长，当前好感度为X.5星】
【某某对你产生仇恨，当前仇恨度为X星】
```

### 人际关系查看
```
【铁老：炼气境七层，对你充满厌恶，仇恨度为1星】
【邢红璇：炼气境七层，对你产生好感，当前好感度为2星】
```

## 五、节奏模式

### 时间跳跃（核心技法）
- **微跳**：`次日一早`、`入夜`、`正午`
- **中跳**：`半年时间过去`、`一晃眼，两年时间`
- **大跳**：`十年转瞬即逝`、`岁月悠悠`
- **格式**：时间跳跃后接 `……` 分隔，再接新场景

### 延迟满足
- 主角等待了11-16年才开始修炼（摇骰子）
- 关键：等待过程中穿插小事件保持趣味
- 满足感在等待结束时爆发

### 章节节奏公式
```
1. 开场（现状/时间跳跃）         ~10%
2. 事件触发（遭遇/发现/系统提示） ~20%
3. 主角反应（内心吐槽 + 分析）    ~30%
4. 行动/选择                      ~25%
5. 结果 + 章末钩子                ~15%
```

### 修为成长节奏
- 前期：境界提升快，增加爽感
- 中期：境界提升减慢，引入更多外部事件
- 后期：闭关 + 时间跳跃 + 偶尔出手震慑

## 六、爽点设计模式

### 1. 扮猪吃虎（核心爽点）
- 敌人小看主角 → 主角秒杀 → 围观者震惊
- 关键：**敌人的震惊反应比战斗过程更重要**
- 示例：`"宗主……还进攻吗？"` → `"进攻？什么进攻！你是不是奸细？我们是来支援玉清宗的！"`

### 2. 属性碾压
- 展示属性面板，让读者直接看到数值差距
- `他才炼气境八层，邢红璇也才炼气境七层`

### 3. 选择奖励
- 系统给出两个选项，主角选低调选项反而获得更好的奖励
- 制造"闷声发大财"的快感

### 4. 好感度上涨
- 配角对主角好感不断上升
- 好感度数字本身带来爽感
- 尤其是仇人变舔狗的反转

### 5. 秒杀式战斗
- 战斗描写极简，1-3段解决
- 重点在战前铺垫（敌人嚣张）和战后反应（敌人呆滞）
- 示例：`三秒后，他睁眼，摇了摇头。就这？`

### 6. 低调失败的喜剧
- 主角想低调 → 魅力/实力不允许 → 被迫高调
- `"我这该死的魅力！"`

## 七、角色设计范式

### 主角模板
| 特质 | 表现 |
|------|------|
| 前世记忆 | 现代人+游戏/网络思维 |
| 核心诉求 | 长生不死 > 变强 > 其他 |
| 行事风格 | 苦修闭关，拒绝冒险 |
| 外表 vs 内心 | 沉默寡言 vs 吐槽满天飞 |
| 对待感情 | 等无敌了再说 |
| 对待麻烦 | 能躲就躲，实在不行秒杀 |

### 配角类型
- **忠厚好友**：被欺负，推动主角出手
- **追求者（女）**：好感度递增，主角躲避
- **师尊/长辈**：实则被主角超越，产生反差
- **反派**：嚣张→震惊→陨落
- **徒弟/下属**：各有奇特来历，闯祸型

### 对话风格
- 修士说话偏古风但不酸（简洁）
- 主角对话很少，大多是内心独白
- 女性角色：`掩嘴笑`、`没好气道`、`幽怨道`
- 反派：狂妄后立刻被打脸

## 八、章尾钩子类型（轮换使用）

| 钩子类型 | 示例 |
|---------|------|
| 悬念型 | 系统显示一个未知信息 |
| 危机型 | 新的强敌出现/暴露风险 |
| 反转型 | 意外发现某人的真实身份 |
| 期待型 | 获得新的奖励/突破即将到来 |
| 搞笑型 | 尴尬/反差的收尾 |
| 新角色型 | 先天气运者/强者登场 |

## 九、世界观搭建方式

### 境界体系（原作）
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 融虚 → 渡劫 → 大乘

### 信息投放方式
- **不集中设定介绍**，随剧情自然展开
- 通过系统面板的 `人际关系` 功能顺带交代
- 通过配角对话透露世界信息
- 每次遇到新角色时附带势力信息

### 地域扩展
- 从宗门药草园 → 宗门内门 → 万妖界 → 大燕修真界 → 更广阔世界
- 随主角修为提升而逐步扩大活动范围

## 十、关键写作禁忌

1. **不要让主角主动找麻烦**——他是被动卷入的
2. **不要写冗长的打斗**——秒杀or几段解决
3. **不要煽情**——用幽默代替沉重
4. **不要让主角说教**——他只想修炼
5. **不要忘记系统面板**——每隔几章展示一次成长
6. **不要让配角比主角更抢镜**——配角是衬托
7. **不要让时间跳跃无事件**——每次跳跃前后都有变化
8. **战斗结束后要有总结反思**——主角会得意或自省

---

## 十一、仿写质量对标

写出的文字应该达到以下效果：
- 读者能笑出来（吐槽要自然不尴尬）
- 读者有"爽"的感觉（碾压+反差）
- 读者想翻下一页（章末钩子有效）
- 不像AI写的（避免工整的排比、对称句式）
- 像一个老练的网文写手在写（口语化、节奏快）
