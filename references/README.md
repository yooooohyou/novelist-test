# 仿写参考资料

存放仿写小说的原始参考作品和相关素材。

## 目录说明

```
references/
├── README.md          ← 本文件
├── <参考作品>.txt      ← 参考小说原文（纯文本）
└── <参考作品>/         ← 如有多个素材可按作品建子目录
    ├── 原文.txt
    └── 笔记.md
```

## 使用方式

1. 将想要仿写的小说原文（.txt）放入本目录
2. 启动仿写工作流时指定参考文件路径，例如：
   > "请分析 `references/顶级气运，悄悄修炼千年.txt` 的写作风格"
3. Phase 1 拆书分析后会在对应小说目录下生成 `02-风格参考手册.md`
